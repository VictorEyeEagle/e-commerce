# E-commerce Website using React, Next.js, TypeScript, and MongoDB

## Overview
This project is an e-commerce website developed using React, Next.js, TypeScript, and MongoDB. It serves as a platform for displaying and selling various products. The MongoDB database is utilized to store product information, which is then fetched and rendered on the frontend. The website features components such as dedicated product pages, a fixed header bar visible across all pages, and a homepage showcasing categories and various products.

## Technologies Used
- React: A JavaScript library for building user interfaces.
- Next.js: A React framework for building server-side rendered and statically generated applications.
- TypeScript: A typed superset of JavaScript that compiles to plain JavaScript.
- MongoDB: A NoSQL database used for storing product information.
- Other Libraries: Various libraries and dependencies for frontend development, including but not limited to routing, styling, and state management.

## Features
### Product Pages
Each product has its dedicated page displaying detailed information such as name, description, price, and images. Users can view product details and make purchasing decisions based on the information provided.

### Fixed Header Bar
The website features a fixed header bar that remains visible at the top of the screen as users navigate through different pages. This header provides easy access to important sections of the website, such as the shopping cart and user account.

### Homepage with Categories and Product Showcase
The homepage serves as a landing page for users and features sections showcasing different product categories. Users can browse through various products and explore different categories based on their interests.

## Setup Instructions
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Set up the MongoDB database and populate it with product information.
4. Start the development server using `npm run dev`.
5. Access the website locally at `http://localhost:3000`.
